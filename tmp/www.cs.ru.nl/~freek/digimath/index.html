This is an overview of systems implementing "mathematics in the computer" as compiled by Freek Wiedijk. Similar information can be found at:

  • Argonne National Laboratory
  • CMU Artificial Intelligence Repository
  • LogBank
  • Stanford Formal Reasoning Group
  • Logic Software and Logic Education

This is the version that is sorted by alphabet. There also are versions with samples and in black-on-white text. And there also are versions of this list grouped by implementation language, by category, by most common interaction mode, by the logic that is supported, and by the size of the effort. Finally, there is a short explanation of the various fields in this database.

This information is still incomplete and there probably are some errors in it. I would appreciate it if people would help me correct and complete it.
